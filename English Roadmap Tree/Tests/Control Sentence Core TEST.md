# Косвенные вопросы


Формула:
[#Formula]`Question word + does + subject + verb?`

1) Where is the company?
   Do you know where the company is?
2) Can you explain why this query runs slowly?
3) Which function causes the error?
   Do you know which function causes the error?
4) What error does this method return?
   Can you tell me what error this method returns?
5) Do you know which database Anna works with?
6) Can you explain why is this report incorrect?
7) Who checks this report?
   Do you know who checks this report?
8) Can you tell me where the developer stores backups?
